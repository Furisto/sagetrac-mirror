Tres Lecturas Sobre Métodos Explícitos En Teoría De Números Usando Sage
=======================================================================

Éste artículo trata sobre el uso de el software matemático Sage para hacer
cálculos con cámpos de números y formas modulares.  Fué escrito para
la reunión de Octubre del 2008 en Burdéos sobre métodos explícitos en teoría de números
(http://www.math.u-bordeaux.fr/gtem2008/).  No asume algún conocimiento prévio sobre Sage,
pero asume una base a nivel de posgrado en teoría algebraica de números.
