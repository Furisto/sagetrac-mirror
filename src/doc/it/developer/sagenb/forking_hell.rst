.. _forking:

==================================================
Fai la tua copia persoanle (Fork) di Sage Notebook
==================================================

Ti basta farlo una volta. Le instruzioni qui sono molto simili alle 
instruzioni a http://help.github.com/forking/ |emdash| vedi quella 
pagina per maggiori dettagli. Stiamo ripetendone una parte qui giusto per dare 
le specifiche per il progetto `Sage Notebook`_ , e per suggerire alcuni nomi di default.

Installa e configura un account su Github
=========================================

Se non hai un account su github, vai alla pagina web di github e fanne uno.

Poi avrai bisogno di configurare il tuo account per permette l'accesso in scrittura 
|emdash| vedi l'help ``Generating SSH keys`` su `github help`_.

Creare la tua copia di fork di `Sage Notebook`_
===============================================

#. Loggati nel tuo accountr github.
#. Vai alla home di `Sage Notebook`_ su github in `Sage Notebook github`_.
#. Fai click sul pulsante *fork*:

   .. image:: forking_button.png

   Ora, dopo una breve pausa, dovresti trovarti nella home page della tua 
   copia di fork di `Sage Notebook`_.
